File Definition include for Patient * Unadmit/Undischarge/Adm & Dsch Date Change Audit patunaaf
Field Type Length Description
DPTUNADM char 8 Admission Number
PTUNDATE char 8 Date of Alteration
PTUNTYPE char 1 Alteration type "1" = Unadmit
"2" = Undischarge
"3" = Change Admission Date
"4" = Change Discharge Date
PTUNODTE char 8 Adm Date or Dsch Date affected
PTUNHOSP char 3 Hospital number for Undischarged patients
DPTUNSEN char 1 Sent flag for EDI (Unadmitted) 0 = No
1 = Yes
PTUNSPAR char 10 Spare variable
lf char 1 Spare Blank
Index Type Segments
1 UNIQUE DPTUNADM PTUNDATE PTUNTYPE
2 UNIQUE PTUNTYPE PTUNDATE DPTUNADM
