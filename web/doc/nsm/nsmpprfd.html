File Def'n Include for Nursing Station Management * Patient Pre-requisite Table nsmppraf
Field Type Length Description
DNSPPADM char 8 Visit number
DNSPPDAY char 3 Day number
DNSPPUNI char 3 Unique Id (order of Pre-requisites)
NSPPTYPE decimal 3 Type of Order 0 = Comments
1 = Medication
2 = Pathology
3 = Radiology
4 = Stock
5 = Special Services
6 = Equipment
7 = Pre-medication
8 = Heading
NSPPDETL char 50 Order Details. This is type depend. 0 -> Free format
1 -> Medication Codes
2 -> Pathology Codes
3 -> Radiology Codes
4 -> Stock code/Quantity
5 -> Department/Description
6 -> Equipment code/Quantity
7 -> Medication Codes
8 -> Free format
NSPPDATE char 8 Date ordered/checked
NSPPTIME char 8 Time ordered/checked
NSPPOPER char 4 Operator who ordered/checked
NSPPPORT char 2 Port who ordered/checked
NSPPSPAR char 30 Spare variable
lf char 1 Spare Blank
