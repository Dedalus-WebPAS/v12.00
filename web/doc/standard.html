Server All
Function Common Inputs & Outputs
Each Business Application Server accepts common input paramters and can output certain standard fields. The fields are identified by the tag %STANDARD.01.xxx[.yyy] where xxx is one of the fields in the table below. The optional yyy paramters are also identified in this table.

Inputs - 01 Standard CGI Parameters

CGI Name Description Table.Field
reportno Option/Report to be processed  
template Business Server Page Template No to be Processed  
securety Form Security Type 0-Not Required
1-Web User ID
2-Web User ID & Password
secureid Form Security ID  
secureid Form Security Password  

Outputs - 01 Standard Ouputs

001-100 Standard Tags
Item Description Table.Field Parameters
001 Printer Selection Options
This will output option tags for printer selection based on a users security. The default printer will be selected based on the last printer the user selected for the report. 
Usage
<select name=schdprnt 
        title="Printer Selection" 
        class=Mandatory>
%STANDARD.01.001.IBAPAT81.01
</select>
  
.xxxxxxxx.yy
xxxxxxx = Report ID 
yy      = Option ID
Used to determine default printer
