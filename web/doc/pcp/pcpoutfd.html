File Design include for Patient Care Plans Patient Outcome Table pcpoutaf
Field Type Length Description
PCOTUNIQ char 10 Unique key (from PCPDIAFD file)
PCOTCODE char 9 Outcome code
PCOTDCOD char 4 Description number
PCOTDATE char 8 Target date (CCYYMMDD)
PCOTTYPE decimal 1 Target type 0 = No target date
1 = By a given date
2 = Number of days
3 = Number of weeks
4 = At all times
PCOTDYWK decimal 3 Target number of days or weeks
PCOTSPAR char 13 Spare variable
lf char 1 Spare Blank
Index Type Segments
1 UNIQUE PCOTUNIQ PCOTCODE
