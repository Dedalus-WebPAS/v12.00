File Def'n Include for Contracting Contracts Assigned (Excluding Waiting List Patients) concasaf
Field Type Length Description
DCOCAADM char 8 Visit Number
DCOCAEPN char 2 Episode Number 0 for non-inpatients & bookings
1+ for inpatient consultant episodes
COCAPROV char 5 Provider Code
COCAPURC char 5 Purchaser Code
COCASERL char 6 Serial Number
COCAEDAT char 8 Episode End Date (CCYYMMDD)
COCACONL char 1 Contract Line Type 1 = Specialty Contract
2 = Specialty/Procedure Contract
3 = General Procedure Contract
COCASPEC char 3 Specialty (Cat A )
COCASERV char 3 Service Type (Cat P )
COCAREF char 9 Contract Line = 1 Emergency Status
E = Emergency Patients Only
N = Non-Emergency Patients Only
Otherwise = All Patients
Contract Line = 2 or 3
Procedure Group Code (CONPGRFD)
COCAVOLU decimal 6 Volume
COCAMANU decimal 1 1 = Manual Allocation, 2 = Auto.
DCOCACTY char 1 Contract Allocation Type 1 = Final Assignment
2 = Provisional Assignment
3 = Prospective Assignment
COCASPR1 char 5 Spare Variable
COCASDAT char 8 Episode Start Date (CCYYMMDD)
COCASITE char 6 Site code (outpatients only)
COCAUFCE char 1 Finished consultant episode? Blank = Yes, "U" = No
COCASPAR char 15 Spare Variable
lf char 1 Spare Blank
