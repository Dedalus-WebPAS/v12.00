Welcome to JavaChart!

This product release contains Java class files, source code files, example applets and documentation.  The directory hierarchy is as follows:

api - JavaChart documentation

    Index
    API Index
chart - JavaChart's chart package

applet - JavaChart's applet package

contrib - useful code samples and utilities

html - HTML files for applet demos and contrib samples

    standard - standard non-interactive demos
    active - standard interactive demos
    contrib - interesting user contributions
utility - a few useful utilities and an interactive application in source code form.

standalone - simple standalone JavaChart demo applications

Out, Damned Spot!  - quick information on removing the JavaChart applet freeware information dialog.
