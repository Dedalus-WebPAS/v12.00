A Few Date/Time Axis Samples

The first sample uses contrived manufacturing data. Note that all these samples obtain data through "customDatasetHandler" because of the variable nature of date stamps.

The next three samples use real data from the US Bureau of Labor Statistics. This data was transferred to Visual Engineering's web server because of Java's restrictions on loading data from foreign servers, but if your system has live statistical information such as that found in this file there's no reason you can't use JavaChart to display that information.
