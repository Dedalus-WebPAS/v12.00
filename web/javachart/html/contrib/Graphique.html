Mode d'emploi:

  • Inscrire jusqu'à un maximum de 16 coordonnées dans le Tableau. Vous pouvez en inscrire moins.
  • Vous devez inscrire un même nombre de valeurs d'ordonnée et d'abcisse.
  • Il est préférable de les inscrire en ordre croissant.
  • Les nombres décimaux sont acceptés avec le point et non la virgule.
  • Les nombres négatifs sont acceptés.
  • La notation scientifique n'est pas gérée par l'application. Vous pouvez toutefois joindre un facteur de multiplication à l'unité de mesure.
  • Appuyez sur le bouton Graphique pour visualiser votre graphique. Il apparaîtra dans une seconde fenêtre.
  • Le serveur ne gardera pas de copie de votre graphique.
