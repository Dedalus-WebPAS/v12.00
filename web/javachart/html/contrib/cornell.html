This class uses a custom dataset handler to read in Axis labels and data sets in a single operation. The data rows follow the param dataRows. The number of columns are calculated from the first row, and is assumed to be uniform throughout. The pipe symbol "|" delineates new rows. The first column is an axis label, subsequent columns represent datasets. Note: This applet behaves differently in appletviewer than in IE and Netscape browsers, due to differences in the behavior of the StringTokenizer class.
