The first example below is generated with random data at 1 second intervals. The data values vary from 0 to 10. Notice the axis scale automatically change to accomodate various datasets: similar data yields tighter grids & smaller axis increments, a dataset with no values over, say, 5.0 will rescale to have a maximum value of 5.0, etc.

This chart also retrieves labels from a URL, so that labels can be varied with the data stream
