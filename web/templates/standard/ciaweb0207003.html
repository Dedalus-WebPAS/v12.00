Export File Generation - %CIAWEB02.07.002
Export Level Export Data File Name
      eg. EXPTFILE

       
The Export file is a standard comma delimited ASCII file with text fields enclosed in double quotation marks. Export files created for levels 1, 2 and 3 will export the casemix codes and descriptions along with the Total Income, Bed Days and Patient Numbers. Export files created at level 4 will contain the casemix code and description for each level, the episode number, patients sex, age, Admission & Discharge date, Bed Days, and Income Details. The file generated may be imported into most standard PC products including Lotus 123 and Microsoft Excel.
